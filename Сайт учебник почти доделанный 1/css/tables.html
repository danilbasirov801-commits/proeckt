📊 Таблицы в HTML

Структурирование данных, теги таблиц, объединение ячеек, стилизация и псевдоклассы. Лектор: Хаятова Лилия Рэхтямовна

📌 1. Основные теги таблицы

HTML-таблица строится с помощью набора тегов: <table>, <caption>, <thead>, <tbody>, <tr> (строка), <th> (ячейка-заголовок) и <td> (обычная ячейка).

<table border="1" cellspacing="0" cellpadding="5">
  <caption>Заголовок таблицы</caption>
  <thead>
    <tr><th>Пункт 1</th><th>Пункт 2</th></tr>
  </thead>
  <tbody>
    <tr><td>Ячейка 1</td><td>Ячейка 2</td></tr>
    <tr><td>Ячейка 3</td><td>Ячейка 4</td></tr>
  </tbody>
</table>
📌 Важно Ячейки <td> всегда должны быть внутри строк <tr>. Количество ячеек в каждой строке должно быть одинаковым (если нет объединения).

💡 Заметка Тег <caption> задаёт заголовок таблицы, он не оборачивается границей и располагается над таблицей (или под ней, в зависимости от CSS).

🔗 2. Объединение ячеек: colspan и rowspan

colspan

Объединяет ячейки по горизонтали (несколько столбцов).

<td colspan="2">Объединённая ячейка</td>
colspan="2"
Столбец 1Столбец 2
12 (объединяет два столбца)
34

rowspan

Объединяет ячейки по вертикали (несколько строк).

<td rowspan="2">Объединённая ячейка</td>
rowspan="2"
Столбец 1Столбец 2
124
(объединяет 2 строки)
3
<!-- Пример сложной таблицы с объединением столбцов и строк -->
<table border="1">
  <tr><th>Пункт 1</th><th>Пункт 2</th><th>Пункт 3</th></tr>
  <tr><td colspan="2">12</td><td>3</td></tr>
  <tr><td rowspan="2">4</td><td>5</td><td>6</td></tr>
  <tr><td>7</td><td>8</td></tr>
</table>

⚠️ Важно При объединении ячеек удаляйте лишние <td> в последующих строках, иначе таблица «поедет».

🎨 3. CSS-стилизация таблиц

С помощью CSS можно улучшить внешний вид таблиц: убрать двойные границы, добавить «зебру», изменить ширину колонок.

🔲 Схлопывание границ (border-collapse)

table { border-collapse: collapse; border: 5px solid gray; }
th, td { border: 2px solid black; padding: 8px; }

🎯 Псевдоклассы для строк и ячеек

ИмяВозрастГород
Анна25Москва
Иван30СПб
Мария22Казань
Петр28Екатеринбург

✨ Применены: tr:nth-child(even) (зебра), tr:hover, td:first-child (жирный стиль первого столбца).

/* Примеры псевдоклассов для таблиц */
tr:nth-child(even) { background-color: #f2e6c9; } /* чётные строки */
tr:nth-child(odd) { background-color: #fff8e7; } /* нечётные */
td:first-child { width: 40%; font-weight: bold; }
td:nth-child(3) { text-align: left; }
tr:hover { background-color: #ffe6b3; }
/* nth-child(3n) — каждый третий элемент, nth-child(3n+2) — со второго и т.д. */
📘 Псевдоклассы: :first-child, :last-child, :nth-child(odd), :nth-child(even), :nth-child(3n), :nth-child(3n+2), :nth-child(3) – позволяют гибко управлять стилями строк и колонок.

📌 Практическое задание

1. Создайте базовую таблицу с заголовком и телом. Используйте теги <thead>, <tbody>, не менее 3 строк и 3 столбцов.

2. Объедините ячейки: в первой строке объедините два столбца с помощью colspan. В первом столбце объедините две строки через rowspan.

3. Стилизация: примените CSS: border-collapse: collapse, добавьте фон для чётных строк, сделайте ширину первого столбца 30%.

4. Создайте таблицу «Прайс-лист» (услуги и стоимость) из 4 столбцов: Услуга, Цена, Срок, Скидка. Используйте псевдоклассы для выделения строк при наведении.

💡 5. Полезные советы по таблицам

  • Атрибуты border, cellpadding, cellspacing устаревают, лучше использовать CSS: border, padding, border-spacing.
  • Для адаптивности на мобильных устройствах таблицы можно оборачивать в <div style="overflow-x:auto;">.
  • Не используйте таблицы для вёрстки макета страницы — это плохая практика. Таблицы нужны только для табличных данных.
  • С помощью :nth-child(3n+1) можно стилизовать каждый третий столбец, начиная с первого.
