📘 Visual Studio Code: редактор кода для веб‑разработки

Мощный, бесплатный редактор от Microsoft для Windows, Linux и macOS. Незаменимый инструмент верстальщика: настройка, расширения, горячие клавиши, лучшие практики.

📁 1. Первые шаги & файловая структура

🔧 Установка и смена языка

Скачать VS Code можно с официального сайта:
code.visualstudio.com/download
Чтобы переключить интерфейс на русский: установите расширение Russian Language Pack (от Microsoft) через меню расширений (Ctrl+Shift+X) и перезапустите редактор.

📂 Правильная структура проекта

  • ✔ Папки должны иметь осмысленные названия: img/ для изображений, css/ для стилей, js/ для скриптов.
  • ✔ Никаких "лишних" файлов в корне – только index.html и основные папки.
  • ✔ Типичная ошибка: изображение не лежит в папке img – избегайте хаоса.
  • ✔ Единый стиль именования файлов: style.css, script.js, понятные названия.

Пример хорошей структуры:

project/
  ├─ index.html
  ├─ css/
  │    └─ style.css
  ├─ img/
  │    └─ logo.png
  └─ js/
       └─ main.js

⚡ 2. Emmet и Live Server – ускорение вёрстки

🚀 Emmet (бывшие Zen Coding)

Позволяет писать сокращения, которые разворачиваются в HTML-код. Установлен в VS Code по умолчанию.

div.container>ul.list>li.item*3 → нажмите Tab — получите структуру: 
<div class="container">
  <ul class="list">
    <li class="item"></li>
    <li class="item"></li>
    <li class="item"></li>
  </ul>
</div>

Больше сокращений: Шпаргалка Emmet

🌐 Live Server — must have для веб-разработчиков

Запускает локальный сервер с автоматическим обновлением страницы при сохранении изменений. Плагин от Ritwick Dey.

  • ✅ Экономит время: не нужно вручную обновлять браузер.
  • ✅ Показывает результат верстки в реальном времени.
  • ▶️ Запуск: ПКМ по HTML-файлу → "Open with Live Server" или кнопка Go Live в правом нижнем углу.

🧩 3. Полезные расширения (плагины) VS Code

🎨 Color Picker – интерактивный выбор цвета, отображение цветовых меток в CSS
✨ Prettier – самоуверенный форматировщик кода (JS, CSS, HTML, JSON)
🌈 Rainbow Brackets – подсветка парных скобок разными цветами
📁 AutoFileName – автоподстановка путей к файлам (img/src и др.)
🔎 CSS Peek – быстрый переход к CSS-правилам из HTML
🖼️ Image Preview – показывает миниатюры изображений на полях в CSS
📐 SVG Viewer – предпросмотр SVG-файлов (ПКМ → SVG Preview)
💡 IntelliSense for CSS – подсказки классов и id в HTML
🚚 AutoImport – автоматический импорт файлов/модулей
💬 Better Comments – улучшенные комментарии (TODO, !, ? и др.)
⏱️ WakaTime – аналитика времени, потраченного на проекты и языки

Совет Устанавливайте расширения через меню Ctrl+Shift+X – ищите по названиям выше. Обязательный минимум: Live Server, Prettier, Rainbow Brackets, Color Picker.

📏 5. Кодстайл и форматирование (Prettier + правила)

Почему важен единый стиль?

Читаемость кода, удобство командной работы, уменьшение ошибок. Используйте Prettier – он автоматически расставляет отступы, переносит длинные строки, выравнивает кавычки.

Настройка: установите Prettier и в настройках редактора включите "Format On Save".

Рекомендации по структуре CSS

  • 1. Позиционирование (position, top, left)
  • 2. Блочная модель (display, margin, padding, border)
  • 3. Типографика (font, text, color)
  • 4. Оформление (background, opacity)
  • 5. Анимации (transform, transition)

Правило хорошего тона: не оставляйте пустые строки внутри селекторов, используйте комментарии для разделов.

🔍 HTML-правила (кратко)

  • ✔ Использовать семантические теги (<header>,<main>,<article>)
  • ✔ Все атрибуты в двойных кавычках
  • ✔ Закрывать все теги, кроме одиночных (<img>, <br>)
  • ✔ Корректная вложенность без пересечений
  • ✔ Добавлять альтернативный текст alt для изображений

📌 6. Практическое задание – закрепляем материал

⚙️ Пошаговая инструкция

  1. Установите VS Code (если еще не сделали).
  2. Переведите интерфейс на русский язык (Russian Language Pack).
  3. Создайте в папке «Документы» проект с именем «Project».
  4. Откройте папку в VS Code (Файл → Открыть папку).
  5. Создайте структуру:
    • файл index.html
    • папку img
    • папку css, внутри файл style.css
  6. Скопируйте в index.html базовый код или любой пример из онлайн-редактора (CodePen).
  7. Установите Live Server и запустите страницу в браузере через правый клик → "Open with Live Server".
  8. Дополнительно установите расширения: Color Picker, Prettier, Rainbow Brackets.
  9. Поэкспериментируйте с Emmet: введите nav>ul>li*5>a и нажмите Tab, чтобы создать навигацию.

🎯 Домашнее задание Изучить возможности горячих клавиш, доустановить AutoFileName и Image Preview, сделать мини-сайт с картинками, стилями и подключить Prettier для автоформатирования.
