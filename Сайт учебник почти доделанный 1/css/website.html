🎨 3. CSS-стили для первого сайта

/* style.css — основные стили для сайта */

body {
  background-color: blueviolet;
}

.title {
  padding: 30px;
  text-align: center;
  font-size: 50px;
}

.title span {
  color: #F9A826;
  font-size: 40px;
}

.main, .main2 {
  display: flex;
  justify-content: center;
  align-items: center;
  flex-wrap: wrap;
}

.main img {
  max-width: 600px;
}

.main-info {
  max-width: 500px;
}

.main-info a {
  font-size: 20px;
  color: #F9A826;
  text-decoration: none; /* отмена подчеркивания */
}

.form {
  max-width: 420px;
  margin: 50px auto;
  display: flex;
  flex-direction: column;
}

.form input, .form textarea {
  padding: 10px;
  font-size: 23px;
  border-radius: 10px;
  margin-bottom: 20px;
  border: 2px solid rgb(175, 171, 171);
  font-family: 'Times New Roman', Times, serif;
}

input::placeholder, textarea::placeholder {
  color: #F9A826;
  font-style: italic; /* курсив */
}

.form button {
  background-color: #F9A826;
  padding: 16px;
  border-radius: 10px;
  font-size: 20px;
  border: 3px solid #F9A826;
  color: white;
  text-transform: uppercase; /* заглавные буквы */
  cursor: pointer;
}

.form button:hover {
  background-color: white;
  color: black;
  border: 3px solid #F9A826;
}

📌 Практическая работа: «Мой первый сайт»

Пошаговая инструкция:

  • Создайте проект «Мой первый сайт_ФИ_П-17» (где ФИ — ваши инициалы, П-17 — номер группы).
  • В проекте создайте папку css и папку img.
  • Создайте файл index.html и файл css/style.css.
  • Скачайте иллюстрацию с undraw.co и сохраните в папку img.
  • Сверстайте главную страницу по образцу из демонстрации выше:
    • Заголовок с вашим именем и должностью (Junior верстальщик)
    • Блок «О себе» с текстом и списком преимуществ
    • Изображение (иллюстрация)
    • Ссылку «Посмотреть мои работы»
    • Форму заявки: имя, телефон, email, многострочное сообщение, кнопка
  • Подключите файл стилей к HTML-документу.
  • Настройте стили по образцу выше (или создайте свои).

✏️ Задание для самостоятельной работы

  1. Поменяйте фон страницы (свойство background-color).
  2. Отмените подчеркивание у ссылки «Посмотреть мои работы» (уже есть в примере — text-decoration: none;).
  3. Добавьте на сайт блок с контактами (телефон, email, соцсети), добавьте иконки в цвет вашего дизайна с flaticon.com, добавьте картинку к этому блоку.
  4. Поменяйте в форме цвет текста, сделайте его курсивом (через font-style: italic;).
  5. Поменяйте в форме тип шрифта на 'Times New Roman', Times, serif.
  6. Сделайте в кнопке текст заглавными буквами через CSS (text-transform: uppercase;).
  7. В форму заявки перед кнопкой добавьте поле многострочного текста (<textarea>) с подсказкой «Ваше сообщение».
  8. Измените стили этого поля, как во всей форме (единый дизайн).

💡 5. Полезные советы

  • Используйте display: flex для удобного расположения блоков.
  • Для изображений всегда указывайте атрибут alt.
  • Проверяйте валидность кода на validator.w3.org.
  • Используйте ::placeholder для стилизации текста-подсказки в полях ввода.
  • Для кнопки добавьте эффект при наведении (:hover).
  • Адаптивность: используйте flex-wrap: wrap и медиа-запросы.
