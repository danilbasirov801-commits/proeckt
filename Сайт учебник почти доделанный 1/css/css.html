🎨 Каскадные таблицы стилей (CSS)

Язык оформления HTML-документов. Лектор: Хаятова Лилия Рэхтямовна

📌 1. Что такое CSS?

CSS (Cascading Style Sheets) — это язык таблиц стилей, который позволяет прикреплять стиль (например, шрифты и цвет) к структурированным документам HTML.

📁 Создание файла CSS

Файл стилей должен иметь расширение .css. Обычно его называют style.css.

style.css
🔗 Подключение CSS к HTML

Для связывания HTML-документа с CSS-файлом используется тег <link> внутри <head>.

<link rel="stylesheet" href="css/style.css">
<!-- Пример подключения внешних стилей и шрифта Google Fonts -->
<head>
  <link rel="stylesheet" href="style.css">
  <link href="https://fonts.googleapis.com/css2?family=Lobster&display=swap" rel="stylesheet">
</head>

💡 Важно Атрибут rel всегда равен stylesheet, а в href указывается путь к CSS-файлу (относительный или абсолютный).

📝 2. Синтаксис CSS

Каждому HTML-тегу соответствует селектор CSS. После селектора ставятся фигурные скобки {}, внутри которых пишутся CSS-свойства.

селектор {
  атрибут: значение;
  атрибут: значение;
}
/* Пример: стили для всех абзацев */
p {
  font-size: 16px;
  color: #333;
}

💬 Комментарии в CSS

/* Это комментарий в CSS */
/* Он не влияет на отображение страницы */
h1 {
  color: green; /* Можно комментировать отдельные свойства */
}

📚 Пример использования классов

HTML
<h1 class="heading">Заголовок</h1>
<p>Обычный абзац</p>
<h2 class="heading">Подзаголовок</h2>
CSS
.heading {
  color: red;
  font-size: 24px;
}

🏷️ Пример использования id

HTML
<h1 id="main-title">Главный заголовок</h1>
CSS
#main-title {
  color: blue;
  text-align: center;
}

⚖️ Вес CSS-селекторов

id имеет больший вес, чем class, а class — больший, чем селектор по тегу.

/* Стиль из id перекроет стиль из class */
#heading { color: blue; } /* применится */
.heading { color: red; }

📏 4. Правила именования классов

  • ❌ Класс не должен начинаться с цифры — .1block нельзя, но .block1 — можно.
  • ❌ Класс не должен содержать русские буквы — .кнопка нельзя, .knopka — можно.
  • ✅ В имени класса допустимо использовать дефис (-) и подчёркивание (_) — .view-design-block и .search_form_button.
  • ✅ Класс можно начинать с дефиса и подчёркивания — .__error__.
  • ⚠️ Классы чувствительны к регистру — .Red, .RED и .red — это разные имена.
/* Правильные имена классов */
.header { }
.main-content { }
.btn_submit { }
._active { }
.card-item-1 { }

🔗 5. Объединение правил для нескольких селекторов

Через запятую
h1, h2, h3 {
  color: green;
  font-family: Arial;
}
Универсальный селектор *
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

💡 Совет Универсальный селектор * применяет стили ко всем элементам на странице. Используйте его осторожно, обычно для сброса отступов.

📌 Практическая работа

Создание проекта «Практика по CSS» в Visual Studio Code

  • Создайте в папке Документы папку Практика по CSS
  • Откройте программу Visual Studio Code
  • Создайте файл index.html
  • Создайте папку css
  • В папке css создайте файл style.css
  • Установите плагин Live Server (если ещё не установлен)
  • В файле index.html нажмите Shift + ! (на английской раскладке), затем Tab — создастся базовая структура HTML
  • Измените <html lang="en"> на <html lang="ru">
  • Замените содержимое <title> на Практика по CSS
  • В тег <body> добавьте абзац с текстом
  • Сохраните изменения (Ctrl+S)
  • В файле style.css напишите стили для абзаца (например, color: red;)
  • Подключите CSS-файл в <head>:
    <link rel="stylesheet" href="css/style.css">
  • Запустите Live Server (правой кнопкой по index.html → "Open with Live Server") и посмотрите результат в браузере
