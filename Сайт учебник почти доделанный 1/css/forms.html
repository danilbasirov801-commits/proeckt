📝 Формы для заполнения

Сбор данных от пользователей: текстовые поля, кнопки, чекбоксы, валидация. Лектор: Хаятова Лилия Рэхтямовна

📌 1. Тег <form> — контейнер для формы

Веб-форма — это специально ограниченная область на странице, куда посетитель может внести информацию и выбрать действия.

🔗 Атрибут action

Содержит адрес, определяющий, куда будет отправлена информация формы.

action="submit.php"
📤 Атрибут method

Определяет, как будет отправлена информация: GET или POST.

method="POST"
<form action="action.php" method="POST" id="form1">
  <input type="text" name="username">
  <input type="submit" value="Отправить">
</form>
💡 Альтернативная отправка
<input type="button" value="Отправить" onClick="btnClick();">

🔘 2. Типы input-элементов

<input type="text"> — текст
<input type="number"> — цифры
<input type="email"> — электронная почта
<input type="tel"> — номер телефона
<input type="password"> — пароль (точки)
<input type="date"> — дата
<input type="search"> — поиск
<input type="file"> — загрузка файла
<input type="hidden"> — скрытое поле
<input type="checkbox"> — флажок
<input type="radio"> — переключатель
<input type="color"> — выбор цвета
<input type="range"> — ползунок
<input type="submit"> — кнопка отправки
<select> — выпадающий список
<textarea> — многострочный текст

📋 3. Пример формы с элементами ввода

💡 Примечание В данной демо-форме отправка не настроена — она предназначена для изучения внешнего вида элементов.

🏷️ 4. Связь label и input

Тег <label> связывает текст с полем ввода. При клике на текст фокус переходит на поле.

🔗 Через id и for
<label for="user-input">Пример ввода</label>
<input type="text" id="user-input" placeholder="Пример ввода">
📦 Вложенный label
<label>
  Пример ввода
  <input type="text" placeholder="Пример ввода">
</label>

⚙️ 5. Важные атрибуты input

🔒 required

Делает поле обязательным для заполнения.

<input type="text" required>
📏 maxlength

Максимальное количество символов.

<input type="text" maxlength="10">
💬 placeholder

Подсказка внутри поля.

<input type="text" placeholder="Введите имя">
📊 min / max

Минимальное и максимальное значение для number, date, range.

<input type="number" min="0" max="100">
📎 accept

Указывает тип файлов для input type="file".

<input type="file" accept="image/*">
✨ autofocus

Автоматический фокус на поле при загрузке страницы.

<input type="text" autofocus>

🔍 6. Паттерны (pattern) — валидация по шаблону

Паттерн — это шаблон формата ввода, основанный на регулярных выражениях.

📞 Телефон
pattern="\+7\-[0-9]{3}\-[0-9]{3}\-[0-9]{2}\-[0-9]{2}"
Формат: +7-xxx-xxx-xx-xx
📅 Дата
pattern="[0-9]{2}\.[0-9]{2}\.[0-9]{4}"
Формат: дд.мм.гггг

✅ Зелёная подсветка — валидный ввод, красная — невалидный.

✅ 7. Валидация форм с псевдоклассами :valid / :invalid

CSS позволяет стилизовать поля в зависимости от их валидности.

/* Стили для невалидных полей */
input:invalid {
  background: rgba(255, 200, 200, 0.5);
  border-color: #ff6b6b;
}

/* Стили для валидных полей */
input:valid {
  background: rgba(200, 255, 200, 0.3);
  border-color: #4caf50;
}

/* Отключить встроенную валидацию браузера */
<form novalidate>...</form>

Поля меняют цвет в зависимости от корректности ввода.

🎯 8. Селекторы атрибутов для стилизации форм

/* Стилизация по типу input */
input[type="text"] {
  border-radius: 8px;
  padding: 10px;
}

input[type="submit"] {
  background: #FF8C00;
  color: white;
}

input[type="submit"]:hover {
  background: #e07a00;
}

📌 Практическое задание

Рассмотрим на практике наиболее популярные теги форм.

  1. Создайте HTML-страницу с формой регистрации, содержащей следующие поля:
    • Имя (text, required)
    • Email (email, required)
    • Пароль (password, minlength="6")
    • Дата рождения (date)
    • Пол (radio: Мужской/Женский)
    • Страна (select с 3-4 вариантами)
    • О себе (textarea)
    • Согласие на обработку данных (checkbox)
    • Кнопка отправки (submit)
  2. Примените стилизацию: добавьте padding, border-radius, изменение цвета при фокусе.
  3. Добавьте паттерн для телефона в формате +7-xxx-xxx-xx-xx.
  4. Используйте псевдоклассы :valid / :invalid для подсветки полей.
  5. Попробуйте атрибут autofocus на первом поле.

✅ Результат: сохраните страницу как register.html и проверьте работу всех элементов формы.
