✍️ Работа с текстом в CSS

Шрифты, размеры, выравнивание, декоративные эффекты, списки и границы. Лектор: Хаятова Лилия Рэхтямовна

📌 1. HTML-теги для форматирования текста

🔖 Выделение и форматирование
<mark>подсветка</mark>
<del>удаленный текст</del>
<s>зачеркнутый</s>
<ins>дополнение</ins>
<u>подчеркнутый</u>
<strong>важный жирный</strong>
<b>жирный</b>
<em>важный курсив</em>
<i>курсив</i>
🔢 Степени и формулы
x<sup>2</sup> → x²
H<sub>2</sub>O → H₂O
💬 Цитаты и время
<q>цитата в тексте</q>
<blockquote>блок с цитатой</blockquote>
<time datetime="2020-08-19T10:00">10:00 19.08.2020</time>
<address>улица Пушкина, дом Колотушкина</address>

Пример: подсвеченный текст, удаленный, зачеркнутый, дополнение, подчеркнутый.

Важный жирный текст, просто жирный, важный курсив, просто курсив.

Математика: x2 + y2 = z2. Химия: H2O и CO2.

Краткая цитата внутри текста.

Блок с цитатой. Здесь может быть длинная цитата или важное высказывание. — источник

15 января 2025 года

г. Москва, ул. Тверская, д. 1

🎨 2. CSS свойства для оформления текста

📏 font-size

Размер шрифта: px, pt, em, rem, %

font-size: 16px; /* 16 пикселей */
🔤 font-family

Тип шрифта (семейство). Важно использовать веб-безопасные шрифты.

font-family: "Arial", "Helvetica", sans-serif;
💪 font-weight

Жирность: bold, normal, 100-900

font-weight: bold;
✒️ font-style

Курсив: italic, normal

font-style: italic;
📐 text-align

Выравнивание: left, right, center, justify

text-align: center;
🎨 color

Цвет текста: имя, HEX, RGB, RGBA

color: #e0b743; color: rgb(255, 0, 0);
📝 text-decoration

Оформление: underline, overline, line-through, none

text-decoration: underline;
🔠 text-transform

Регистр: uppercase, lowercase, capitalize

text-transform: uppercase;

📏 line-height и text-indent

📏 line-height

Межстрочный интервал. Рекомендуется 1.5 для удобочитаемости.

line-height: 1.5;
📝 text-indent

Красная строка (отступ первой строки).

text-indent: 20px;

🎯 Свойство-сокращение font

/* Порядок: курсив жирность размер_шрифта / line-height семейство_шрифта */
font: italic bold 16px/1.5 "Arial", sans-serif;

🔤 3. Семейства шрифтов и Google Fonts

📚 Основные семейства
  • serif — с засечками (Times New Roman)
  • sans-serif — без засечек (Arial, Helvetica)
  • cursive — курсивные (Comic Sans MS)
  • fantasy — декоративные
  • monospace — моноширинные (Courier New)
🌐 Google Fonts

Бесплатные веб-шрифты от Google. Подключаются через <link> или @import.

<link href="https://fonts.googleapis.com/css2?family=Lobster&display=swap" rel="stylesheet">
https://fonts.google.com/

serif — шрифт с засечками

sans-serif — рубленый шрифт

monospace — моноширинный шрифт

📦 5. Границы (border) и скругления (border-radius)

📐 Свойства границы
  • border-width — толщина
  • border-color — цвет
  • border-style — тип: solid, dotted, dashed, ridge, double
border: 2px solid #b45f06;
🔄 Скругление углов
border-radius: 10px; /* все углы */
border-radius: 10px 20px 5px 15px;
border-radius: 20px 10px;
Пример скруглённых углов
/* Границы для отдельных сторон */
border-left: 3px solid red;
border-right: 2px dashed blue;
border-top: 1px dotted green;
border-bottom: 4px double orange;

🖼️ 6. Фоновые свойства

🎨 background-color

Заливка фона одним цветом.

background-color: #f5e5c0;
🖼️ background-image

Фоновая картинка.

background-image: url("bg.png");
🔁 background-repeat

Повторение: no-repeat, repeat-x, repeat-y

📍 background-position

Позиция: top, bottom, left, right, center

📋 7. Списки: ul, ol, dl

🔘 Неупорядоченный список (ul)
<ul type="disc">
  <li>чай</li>
  <li>кофе</li>
</ul>
  • чай
  • кофе
  • какао
🔢 Упорядоченный список (ol)
<ol type="A" start="3">
  <li>первый</li>
</ol>
  1. пункт A3
  2. пункт A4
📖 Список определений (dl)
<dl>
  <dt>HTML</dt>
  <dd>язык разметки</dd>
</dl>
HTML
язык гипертекстовой разметки
CSS
каскадные таблицы стилей

📌 Стилизация списков (list-style)

ul { list-style: square outside; }
ol { list-style: upper-roman inside; }
/* list-style-type: disc, circle, square, decimal, lower-alpha и др. */

🔗 Подробнее: htmlbook.ru/css/list-style

📏 8. Единицы измерения в CSS

📐 Абсолютные

px (пиксели), cm, mm, pt (пункты)

font-size: 38px; width: 500px;
📊 Относительные

em (относительно родителя), rem (относительно корня), % (проценты)

font-size: 2em; width: 50%;

  5. Добавьте границу с закруглёнными углами для блока с текстом.
