🌐 1. Языки для создания сайтов

Чтобы создать современный сайт, веб-разработчик использует несколько языков. У каждого из них своя задача.

📄 HTML & CSS Вёрстка — размещение элементов сайта по нужным местам. HTML создаёт каркас, CSS отвечает за оформление.
⚙️ PHP Программирование (серверная логика). Пример: регистрация пользователей, обработка форм.
✨ JavaScript Оживление сайта: слайдеры, интерактивные карты, анимации.
<!-- HTML строит каркас, CSS красит, JS добавляет интерактивность -->

🏗️ 2. Язык HTML — основа веб-страниц

HTML (HyperText Markup Language) задаёт структуру документа: заголовки, абзацы, списки, изображения. Это скелет любой страницы.

🧱 Структура HTML-документа

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Заголовок страницы (вкладка браузера)</title>
    <!-- ссылки на CSS, мета-теги, иконки -->
  </head>
  <body>
    Основное содержимое: текст, картинки, ссылки...
  </body>
</html>
<head> Содержит метаданные: <title>, кодировку, ссылки на CSS, информацию для SEO.
<body> Всё, что видит пользователь: текст, изображения, кнопки. Обязательный элемент, используется один раз.

📌 Важно Большинство тегов парные: <p>...</p>. Закрывающий тег содержит слеш /. Есть и непарные (одиночные): <img>, <hr>, <br>.

📝 3. Форматирование текста

H1–H6 Заголовки: от самого важного (h1) до подзаголовков (h6).
<p> Абзац текста.
<strong> Логическое выделение важности (жирный).
<em> Логическое акцентирование (курсив).
<b> / <i> Физическое жирное/курсивное начертание без смысловой нагрузки.
<h1>Главный заголовок</h1>
<p>Это обычный абзац с <strong>важной фразой</strong> и <em>акцентом</em>.</p>
<p>Текст может быть <b>жирным</b> или <i>курсивом</i> без дополнительной семантики.</p>

📋 Списки: упорядоченные и неупорядоченные

ul Неупорядоченный (маркированный) список:
<ul>
  <li>Пункт 1</li>
  <li>Пункт 2</li>
</ul>
ol Упорядоченный (нумерованный) список:
<ol>
  <li>Первый шаг</li>
  <li>Второй шаг</li>
</ol>

💬 4. Комментарии в HTML

Комментарии не отображаются на странице, но видны в исходном коде. Используются для пояснений или временного отключения кода.

<!-- Это комментарий. Браузер его проигнорирует -->
<p>Видимый текст</p>
<!-- <p>Закомментированный абзац</p> -->

💡 Зачем? Оставлять заметки себе или коллегам, отлаживать код, временно скрывать блоки.

🔗 5. Ссылки и изображения

Гиперссылки (тег <a>)

<a href="https://html5book.ru">Справочник по HTML</a>
<a href="page2.html">Перейти на другую страницу сайта</a>

🖼️ Изображения (тег <img>)

<img src="https://code.mu/smile.png" alt="Улыбающийся смайлик">
<img src="images/logo.jpg" width="200" height="100" alt="Логотип">

Важно Атрибут alt обязателен: описывает картинку (для SEO и скринридеров).

📸 Ссылки в виде картинок

<a href="https://phphtml.net" target="_blank">
  <img src="https://code.mu/smile.png" alt="Ссылка-картинка">
</a>

Такая конструкция превращает картинку в кликабельный элемент.

🛠️ 6. Инструменты для создания HTML-кода

Для начала можно использовать онлайн-редакторы, например CodePen. Это простая среда, где сразу виден результат вёрстки.

✏️ CodePen Откройте редактор: https://codepen.io/pen/
В окне HTML пишете код, справа — предпросмотр в реальном времени. Идеально для экспериментов.
<!-- Пример кода для CodePen -->

<p>Это моя первая веб-страница.</p>

Также можно использовать обычный блокнот или любой редактор кода (VS Code). Главное — сохранять файлы с расширением .html и открывать в браузере.

📄 7. Базовая страница и спецсимволы (CER)

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Это заголовок тайтл</title>
  </head>
  <body>
    Это основное содержимое страницы.
    <p>Символы: &lt; — <, &copy; — ©</p>
  </body>
</html>

🔣 Специальные символы (Character Entity Reference): &lt; → <, &copy; → ©, &nbsp; — неразрывный пробел.

Дополнительные справочные материалы: html5book.ru/html-html5/

📌 Практическое задание

1. Откройте онлайн-редактор CodePen: https://codepen.io/pen/

2. Воспроизведите базовую структуру HTML-документа:

<!DOCTYPE html>
<html>
<head><meta charset="utf-8"><title>Моя первая страница</title></head>
<body>
  <h1>Добро пожаловать!</h1>
  <p>Это мой первый абзац.</p>
  <!-- добавим список -->
  <ul><li>HTML</li><li>CSS</li><li>JavaScript</li></ul>
  <img src="https://code.mu/smile.png" alt="смайлик">
</body>
</html>

3. Добавьте на страницу: заголовки h2, h3, абзац с <strong> и <em>, упорядоченный список (ol), ссылку на любой сайт и картинку-ссылку.

4. Поэкспериментируйте с комментариями: закомментируйте часть кода и посмотрите, как меняется отображение.

🎯 Открыть CodePen и начать
