🎨 5. Работа с макетом в Figma

📐 Регистрация

Зарегистрируйтесь на сайте figma.com — это бесплатно.

📥 Скачивание макета

Скачайте файл Melody.fig (дизайн-макет сайта) из папки «16_Верстка ЖК Мелодия».

🔍 Инспектирование элементов в Figma

Выделите любой элемент макета → во вкладке Inspect (справа) увидите точные CSS-свойства: цвет фона, размер шрифта, отступы, ширину.

Пример: background: #635854; /* цвет фона шапки */

📌 Практическая работа по верстке «ЖК Мелодия»

Цель: Сверстать шапку сайта в точном соответствии с макетом Figma.

📋 Пошаговая инструкция

  • Переименуйте <title> на ЖК Мелодия.
  • Скачайте файл normalize.css и подключите его в HTML после title.
  • Зарегистрируйтесь на Figma и скачайте макет Melody.fig.
  • В макете изучите структуру шапки: логотип, навигационная панель, контактная информация.
  • Создайте в body шапку сайта: <header class="header">.
  • Внутри header создайте <div class="navbar">.
  • Добавьте логотип (кликабельный <a> с картинкой). Скачайте логотип из Figma в формате SVG в папку img.
  • Создайте навигационную панель — список <ul> со ссылками.
  • Добавьте контактный блок: <span> с текстом и ссылка на телефон.
  • В файле style.css пропишите стили:
    • Цвет фона шапки: #635854
    • Flexbox для navbar (display: flex; justify-content: space-between; align-items: center;)
    • Уберите маркеры у списка (list-style: none)
    • Уберите подчеркивание у ссылок (text-decoration: none), цвет ссылок — белый
    • Добавьте отступы элементам навигации (gap: 30px)
  • Подключите шрифт Roboto с Google Fonts.
  • Задайте внутренние отступы для header (padding).
  • Добавьте полоску (border-bottom) для стилизации.
  • Добавьте в начало стилей * { box-sizing: border-box; }.
  • Напишите медиа-запросы для адаптивности (на мобильных устройствах меню становится вертикальным).

✅ Результат работы — готовая шапка сайта, соответствующая дизайну из Figma. Дальнейшая вёрстка основной части и подвала по макету.
