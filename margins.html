📏 Отступы в CSS

Модель блока: margin (внешние отступы) и padding (внутренние). Работа с маркерами списков. Лектор: Хаятова Лилия Рэхтямовна

📦 1. Box Model (блочная модель)

Каждый HTML-элемент представляет собой прямоугольник, который состоит из четырёх слоёв:

Margin (внешний отступ) — прозрачная область вокруг границы
Border (граница)
Padding (внутренний отступ)
Content (содержимое)
/* Блочная модель элемента */
.element {
  margin: 20px; /* внешние отступы */
  border: 2px solid black; /* граница */
  padding: 15px; /* внутренние отступы */
  width: 300px;
}
💡 Свойство box-sizing

box-sizing: border-box; — ширина включает padding и border, что упрощает расчёты.

* { box-sizing: border-box; } /* рекомендуется для всех элементов */

⬅️➡️ 2. Margin — внешние отступы

Отступы между элементами. Задаются для каждой стороны отдельно или сокращённо.

📐 Способы задания
/* Раздельно по сторонам */
margin-top: 10px;
margin-right: 20px;
margin-bottom: 10px;
margin-left: 20px;

/* Сокращённая запись */
margin: 10px 20px; /* верх/низ, право/лево */
margin: 10px 20px 15px 25px; /* верх, право, низ, лево */
margin: 10px; /* все стороны одинаково */
🎨 Визуальный пример
Блок с margin: 30px (отступ от соседних элементов)

Между этим блоком и другими элементами будет расстояние 30px.

/* Центрирование блока по горизонтали */
.center-block {
  width: 200px;
  margin: 0 auto;
}

🖌️ 3. Padding — внутренние отступы

Расстояние от содержимого до границы элемента. Увеличивает общий размер блока.

📐 Способы задания
/* Раздельно по сторонам */
padding-top: 10px;
padding-right: 20px;
padding-bottom: 10px;
padding-left: 20px;

/* Сокращённая запись */
padding: 15px; /* все стороны */
padding: 10px 20px; /* верх/низ, право/лево */
🎨 Визуальный пример
Блок с padding: 30px (внутренний отступ)

Текст внутри блока отстоит от границы на 30px.

⚠️ Важно

Если не задан box-sizing: border-box, то padding увеличивает общую ширину/высоту элемента.

/* Без border-box: ширина = 200px + 20px padding */
.block { width: 200px; padding: 10px; } /* реальная ширина 220px */

📋 4. Работа с маркерами списков (list-style)

CSS предоставляет множество возможностей для оформления маркеров списков.

🎯 list-style-type
  • square — маркер в виде квадрата
  • circle — маркер в виде незакрашенного круга
  • none — маркер не отображается
  1. lower-roman — римские цифры (строчные)
  2. второй пункт
  1. upper-roman — римские цифры (заглавные)
  1. lower-alpha — строчные буквы
  2. второй пункт
  1. upper-alpha — заглавные буквы
  1. decimal — десятичные цифры (по умолчанию)
🖼️ list-style-image

Использование изображения в качестве маркера.

ul {
  list-style-image: url("li.gif");
}
  • Маркер в виде оранжевого круга
  • Второй пункт списка
📝 list-style-position
ul { list-style-position: inside; } /* маркер внутри блока */
ul { list-style-position: outside; } /* маркер снаружи (по умолчанию) */
/* Свойство-сокращение list-style */
ul {
  list-style: square inside url("li.gif");
}

🔗 Подробнее: htmlbook.ru/css/list-style

📌 Практическое задание

Создание проекта «Отступы_ФИ_П-15»

  1. Создайте в папке Документы папку Отступы_ФИ_П-15 (где ФИ — ваши инициалы).
  2. Откройте программу Visual Studio Code.
  3. Создайте файл index1.html и файл style1.css.
  4. В HTML-файле создайте структуру документа и подключите CSS.
  5. Создайте блок с id="green" (например, div или section).
  6. Добавьте внешние отступы (margin) для этого блока:
    #green {
      margin-top: 20px;
      margin-right: 30px;
      margin-bottom: 20px;
      margin-left: 30px;
      background-color: #90EE90;
      padding: 15px;
    }
  7. Протестируйте разные значения margin и padding, наблюдая за изменениями.
  8. Добавьте на страницу несколько списков с разными типами маркеров (list-style-type).
  9. Попробуйте использовать list-style-image с произвольной картинкой.

✅ Результат: сохраните проект и продемонстрируйте работу всех отступов и стилей списков.
